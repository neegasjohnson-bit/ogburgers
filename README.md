# Promo Hamburgueria

Este projeto é uma aplicação web para uma hamburgueria, permitindo que os usuários montem suas promoções de hambúrgueres e façam pedidos online.

## Estrutura do Projeto

O projeto possui a seguinte estrutura de diretórios:

```
promo-hamburgueria
├── src
│   ├── html
│   │   └── index.html        # Estrutura HTML da aplicação
│   ├── css
│   │   └── styles.css        # Estilos CSS da aplicação
│   └── js
│       └── app.js            # Lógica JavaScript da aplicação
├── package.json               # Configuração do npm
├── .gitignore                 # Arquivos a serem ignorados pelo Git
└── README.md                  # Documentação do projeto
```

## Instalação

1. Clone o repositório:
   ```
   git clone <URL_DO_REPOSITORIO>
   ```
2. Navegue até o diretório do projeto:
   ```
   cd promo-hamburgueria
   ```
3. Instale as dependências (se houver):
   ```
   npm install
   ```

## Uso

1. Abra o arquivo `src/html/index.html` em um navegador para visualizar a aplicação.
2. Siga as instruções na interface para montar sua promoção de hambúrgueres e finalizar seu pedido.
